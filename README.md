# Radar Frequency Clustering using K-Means

This project analyzes radar frequency datasets and groups similar frequency ranges using the K-Means clustering algorithm.

## Features
- Load radar frequency data from CSV
- Apply K-Means clustering
- Generate frequency range clusters
- Visualize cluster distribution

## Technologies
- Python
- Machine Learning
- K-Means Clustering
- Data Visualization

## Workflow
Radar Data (CSV) → K-Means Algorithm → Frequency Clusters → Visualization

## Output
The system produces clustered frequency ranges and visual charts for analysis.
